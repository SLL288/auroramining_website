# Aurora Mining Services Homepage

React + Tailwind CSS homepage for Aurora Mining Services.

## Local Development

```bash
npm install
npm run dev
```

Open the local URL shown by Vite, usually:

```text
http://127.0.0.1:5173/
```

## Production Build

```bash
npm run build
```

The deployable files will be generated in `dist/`.

## Preview Production Build

```bash
npm run preview
```

## Assets

The company logo is loaded from:

```text
assets/company-logo.png
```

The current mining-site image is a placeholder project asset and can be replaced with real field photography later:

```text
assets/aurora-mining-hero.png
```
